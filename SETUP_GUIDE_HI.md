# Family Referral System - Setup Guide (Hindi)

## Kya Badla Hai?

Aapke family management app mein ab ek referral code system add ho gaya hai. Ab naye users sirf referral code ke through hi family mein join kar sakte hain.

## Admin Ke Liye Steps

### 1. Pehli Baar Login
- Jab aap pehli baar login karenge, aapko family setup page dikhega
- Apni family ka naam enter karein (jaise "Sharma Family")
- "Create Family" button click karein
- Aapko ek 8-character referral code milega (jaise: ABC12XYZ)
- Yeh code save kar lein!

### 2. Referral Code Kaise Dekhen
- Dashboard → Settings page par jaayein
- "Family Referral Code" section mein aapka code dikhega
- "Copy" button se code copy kar sakte hain
- Yeh code family members ke saath share karein

### 3. Members Ko Invite Kaise Karein
1. Settings se apna referral code copy karein
2. WhatsApp/Email/SMS se family members ko bhejein
3. Unhe signup page ka link bhejein
4. Members signup karte waqt yeh code enter karenge
5. Aap dashboard se unhe approve kar sakte hain

## New Members Ke Liye Steps

### 1. Registration
- Signup page par jaayein
- **Referral Code** field mein admin se mila code enter karein
- Apna naam, email, aur password enter karein
- "Create account" button click karein

### 2. Approval Wait
- Registration ke baad aapko "Pending Approval" message dikhega
- Admin jab approve karega, tab aap login kar paayenge
- Email check karte rahein approval notification ke liye

### 3. Login
- Approval ke baad login page par jaayein
- Apna email aur password enter karein
- Dashboard access ho jayega

## Important Points

✅ **Mandatory Referral Code**: Bina code ke registration nahi ho sakti
✅ **One Family Per Code**: Ek code se sirf ek family join ho sakti hai
✅ **Admin Control**: Sirf admin hi members ko approve kar sakta hai
✅ **Secure**: Sirf code holders hi family mein aa sakte hain

## Troubleshooting

### "Invalid referral code" Error
- Code sahi se type kiya hai ya nahi check karein
- Spaces ya extra characters na ho
- Admin se confirm karein ki code sahi hai

### "Pending Approval" Stuck
- Admin se contact karein
- Admin ko Members page par jaake approve karna hoga

### Referral Code Nahi Dikh Raha
- Sirf admin ko code dikhta hai
- Settings page check karein
- Agar nahi dikha toh logout/login try karein

## Support

Koi problem ho toh admin se contact karein ya technical support se baat karein.
